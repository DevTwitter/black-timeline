# مشارکت در پروژه

اگر اطلاعات جدیدی دارید یا موردی نیاز به اصلاح دارد:

- Repository را Fork کنید
- تغییرات خود را داخل `timeline.md` اعمال کنید
- یک Pull Request باز کنید

یا:

- از بخش Issues یک Issue جدید ثبت کنید
- اطلاعات و منابع مرتبط را ارسال کنید

---

## قوانین

- برای هر رویداد حداقل یک منبع معتبر قرار دهید
- فرمت فایل را تغییر ندهید
- متن‌ها کوتاه، شفاف و بدون توهین یا جهت‌گیری باشند
- در صورت عدم اطمینان، موضوع را داخل توضیحات ذکر کنید

---

## فرمت هر رویداد

```md
## سال — عنوان

- وزیر: نام وزیر یا —
- توضیحات:
  - متن
  - متن

- منابع:
  - https://example.com
```
